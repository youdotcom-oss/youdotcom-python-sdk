# Search
(*search*)

## Overview

### Available Operations

* [unified](#unified) - Returns a list of unified search results from web and news sources

## unified

Returns a list of unified search results from web and news sources

### Example Usage

<!-- UsageSnippet language="python" operationID="get_/v1/search" method="get" path="/v1/search" -->
```python
import os
from youdotcom import You


with You(
    api_key_auth=os.getenv("YOU_API_KEY_AUTH", ""),
) as you:

    res = you.search.unified(query="Your query")

    # Handle response
    print(res)

```
